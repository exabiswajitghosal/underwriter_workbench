Data Extraction API

Fetch Email Data

URL: {{host}}/api/fetch_email

Method: GET

Description: Fetches the email data.

Parameters:

  • submission_id (required): The ID of the submission to generate data from.

Example Request:

GET /api/fetch_email

Example Response:

{
    "submission_id": "example_submission_id"
}

Response Codes:

  • 200 - OK (Email data fetched successfully)
  • 400 - Bad Request (No email received)

Generate Data

URL: {{host}}/api/generate_data

Method: GET

Description: Generates content from documents based on the provided submission ID.

Parameters:

  • submission_id (required): The ID of the submission to generate data from.

Example Request:

GET /api/generate_data?submission_id=example_submission_id

Example Response:

{
    "generated_content": "example_generated_content"
}

Response Codes:

  • 200 - OK (Data generated successfully)
  • 404 - Not Found (Submission ID not provided or data not found)
  • 500 - Internal Server Error (Unable to generate response)
